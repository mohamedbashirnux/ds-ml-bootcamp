# Project Title
**AI-Based Business Bankruptcy Prediction System Using Financial Ratios**

# Short Description
This project is an end-to-end Machine Learning pipeline that predicts corporate bankruptcy risk by analyzing 95 financial ratios from the Taiwan Economic Journal dataset (Kaggle). It compares the performance of Logistic Regression and Random Forest algorithms to accurately classify risk levels. For practical deployment, the trained model is integrated into a Flask REST API connected to an intuitive web dashboard, enabling real-time risk assessment from user-provided financial inputs.
