# AI-Powered Disease Prediction System

## Project Title
Disease Predictor - AI-Based Medical Diagnosis Assistant Using Machine Learning

## Short Description
An end-to-end Machine Learning system that predicts diseases based on patient symptoms with 94%+ accuracy. The system analyzes 123 symptoms to predict from 53 different diseases including cancers, heart disease, diabetes, HIV/AIDS, and respiratory conditions. It compares Random Forest and Naive Bayes algorithms to provide top-K ranked predictions with confidence scores. The trained models are deployed via a Flask REST API with an interactive web dashboard for real-time disease prediction.
